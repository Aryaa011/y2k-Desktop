
🖥️ AryaOS — Y2K Desktop Experience
A nostalgic, fully interactive desktop interface built with vanilla HTML, CSS, and JavaScript — inspired by Windows XP and Windows 98. Runs entirely in the browser with zero dependencies.

✨ Features
🗂️ Desktop

Draggable desktop icons
Right-click context menu (long-press on mobile)
Sticky Notes — drag them anywhere, type anything
Wallpaper Changer — 3 pixel/Y2K-style wallpapers
CRT Filter toggle — scanlines + vignette over the whole screen
Clock with full date tooltip on hover

🪟 Window System

Open, close, minimize, and maximize windows
Taskbar with active window tabs
Start Menu with Programs submenu
Chaos Mode — random retro error popups

📦 Built-in Apps
AppDescription📝 NotepadType, count chars/lines, download as .txt🔢 CalculatorFull arithmetic, expression display, operator memory💬 MSN MessengerAI bot replies in Y2K AIM-speak🐍 SnakeArrow keys + tap buttons, score/hi-score tracking🎨 PaintPencil, eraser, fill bucket, shapes, brush size, undo (20 steps), save as PNG🎵 Music PlayerPlay/pause/seek with retro UI📷 CameraWebcam capture with snapshot🃏 SolitaireDrag-and-drop card game💻 My ComputerRetro file explorer UI🌐 Internet ExplorerGlitch browser with ERR_NOSTALGIA_OVERFLOW♻️ Recycle BinIt's empty (as it should be)

📱 Mobile Support
AryaOS is fully responsive and works on phones and tablets:

Windows open full-screen on mobile automatically
Desktop icons wrap into a tap-friendly grid
Taskbar tabs scroll horizontally
Snake controls use large on-screen buttons
Calculator buttons are tap-sized (44px)
Long-press the desktop to open the context menu
MSN input uses 14px font to prevent iOS zoom
Landscape and portrait modes both handled


🛠️ Tech Stack

HTML5 — semantic structure
CSS3 — retro XP styling, gradients, animations, full responsive layout
Vanilla JavaScript — all interaction logic, no frameworks, no build step
