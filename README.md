
💿 Y2K Desktop Experience

A nostalgic, interactive desktop interface inspired by early 2000s operating systems like Windows XP and Windows 98.

✨ Features

- 🖥 Draggable desktop windows
- 🎵 Built-in music player
- 🎨 Paint-style drawing app
- 📸 Camera integration
- 💿 Retro CD player UI
- 🧠 Interactive elements and playful UI behavior
- 🎀 Y2K aesthetic with pastel gradients and pixel-style design

---

🎮 Experience

This project recreates the feeling of early digital interfaces — blending nostalgia with modern web interactivity.

Users can:

- Open and move windows
- Interact with mini apps
- Experience layered, chaotic desktop UI

---

🛠 Tech Stack

- HTML
- CSS (gradients, retro styling, animations)
- JavaScript (interaction logic, draggable UI, media handling)
